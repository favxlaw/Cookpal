How-to Use CookPal

With these three easy steps, you can find, select, and start cooking delicious meals with CookPal.

Search for a Recipe

Use CookPal's search feature to find the
perfect recipe for your needs.
Enter keywords such as
the name of the dish and ingredients.

Select a Recipe

Once you find a recipe you like, select it to
see a detailed view of the recipe. The
detailed view will show you the ingredients,
cooking steps, and any special instructions.

Start Cooking

When you're ready to start cooking, CookPal
will show you step-by-step instructions
to cook the Recipe.
